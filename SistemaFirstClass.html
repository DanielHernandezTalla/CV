Sistema Fisrt Class

Descripción

El Sistema First Class es un proyecto desarrollado para ayudar en la administración en un salón de eventos, haciendo más fácil distintas tareas como:

  • La organización de eventos, pasando de hacer todo en físico a hacerlo en un software dedicado.
  • Fácil impresión de detalles de cada evento.
  • Creación de documentos para pago a los meseros y demás empleados

El programa fue creado para estar corriendo en local, pero también puedes poner la base de datos en un servidor y no habría ningún problema.

Instalación

El Sistema First Class no requiere una instalación, ya que este es un programa hecho con tecnología web y por lo tanto con solo tener el ejecutable es suficiente.
