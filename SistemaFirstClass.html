Sistema Fisrt Class

Descripción

El sistema first class es un proyecto desarrollado para ayudar en la administracion en un salon de eventos, haciendo mas facil distintas tareas como:

  • La organizacion de eventos, pasando de hacer todo en fisico a hacerlo en un software dedicado.
  • Facil impresion de detalles de cada evento.
  • Creacion de documentos para pago a los meseros y demas empleados

El programa fue creado para estar corriendo un local, pero tambien puedes poner la base de datos en un servidor y no habria ningun problema.

Instalacion

El sistema first class no requiere una instalacion, ya que este es un programa hecho con tecnologia web y por lo tanto con solo tener el ejecutable es suficiente.
